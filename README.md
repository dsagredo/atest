- react-redux
- bootstrap
- axios
- PokeAPI -> [PokeAPI](https://pokeapi.co)

## Installation
```sh
$ git clone https://github.com/trionoputra/pokemon-app.git
$ cd pokemon-app/
$ npm install
$ npm start
```
